Шартты операторлар өте маңызды, себебі олар шарттың нәтижесіне байланысты әртүрлі кодты орындауға мүмкіндік береді. C++ тілінде тексеруге мүмкіндік беретін үш құрылым бар.


Егер - else құрастырыңыз

егер басқа болса арқылы бір немесе бірнеше шарттарды тексеруге болады, егер олар сәтті тексерілсе, біреуі орындалады, әйтпесе екіншісі.


Мысалды қарастырайық:

int a = 2, b = 10;
        if (a == b) { // Если a будет равным b, тогда будет выполнен код
            // Здесь код что будет выполнен
            // Если все одна строка кода, то фигурные скобки {}
            // можно не ставить
        } else if (a <= b) { // Если а будет меньшим или равным b
            // Если предыдущее условие не будет выполнено,
            // а здесь условие окажется верным,
            // то будет выполнен этот код
        } else {
            // Этот код сработает, если другие условия не будут выполнены
        }

Құрылымды тек бір if шартымен жазуға болады немесе оған қалауыңызша көптеген кірістірілген else if шарттарын қосуға болады.


Әрбір шарттың ішінде жаңа өрнектерді сынайтын басқа if else конструкцияларын жаза аласыз.


Бір мәлімдемеде бірнеше шартты тексеру қажет болса, логикалық «және» немесе логикалық «немесе» қолдануға болады:

if (a != b && a > b) {
            // Код будет выполнен, если и первое, и второе условие
            // окажутся верными
        }
        
        if (a < b || a == b) {
            // Код будет выполнен, если или первое, или второе условие
            // окажется верным
        }

Үштік операция

Бұл if other сөзінің стенографиялық түрі. Мынадай жазылған:
int x = 90, a = 8;
        int res = x < a ? (x + a) : (x - a);
        cout << res << endl;
Тексеріңіз: егер “x” “a” мәнінен аз болса, олардың қосындысын res айнымалысына қойыңыз, әйтпесе – олардың айырмашылығы. 

Ауыстыру

жағдай құрылымында мән сәйкестігі үшін бірнеше шарттарды тексеруге ыңғайлы пішім бар. Айнымалы құрылымға, тексерілетін нәрсеге, сондай-ақ тексеру орындалатын мәндерге жазылады.


Пример оператора:

int x = 23;
        switch (x) { // Проверяем переменную x
            case 1: // Если переменная будет равна 1, то здесь сработает код
                // Может быть множество строк, а не только одна
                cout << "Переменная равна 1" << endl;
                break; // Указываем конец для кода для этой проверки
            case 56: // Если переменная будет равна 56, то здесь сработает код
                // Может быть множество строк, а не только одна
                cout << "Переменная равна 56" << endl;
                break; // Указываем конец для кода для этой проверки
                // По аналогии таких проверок может быть множество
                // Также можно добавить проверку, которая сработает в случае
                // если все остальные проверки не сработают
            default:
                cout << "Что-то другое" << endl;
                break; // Можно и не ставить, так как это последние условие
        }
