Достық сыныптары дос функцияларына ұқсас жұмыс істейді. Жалғыз ерекшелік - біз тек функцияларды ғана емес, бүкіл сыныптарды біріктіреміз.


Dostu қасиетінің арқасында біз бір-бірімен байланысты болатын бірнеше сыныптарды белгілей аламыз. Бұл бір уақытта бірнеше сыныптағы нысан арқылы деректерді алуға мүмкіндік береді.


Осыған байланысты біз ойындағы әртүрлі таңбалардың немесе қолданбадағы терезелердің логикасын дәл сипаттай аламыз. Олардың барлығы әртүрлі сыныптарда жазылуы мүмкін, бірақ сонымен бірге әркім бір-біріне қол жеткізе алады.


Дос класын құру мысалы:

class Auto;
        class Motorcycle {
            friend class Auto;
        private: 
            bool isStillWorking = true;
        };
        
        class Auto {
        public:
            void CrashDTP(Auto &moto) {
                moto.isStillWorking = false;
                cout << "Motorcycle is currently not working!";
            }
        };

Мысалы дос сыныптары дос функцияларына ұқсас құрылымды пайдаланатынын көрсетеді.


Мотоцикл сыныбының мәлімдемесі алдында Auto класының бар екендігі көрсетіледі: class Auto. Содан кейін біз Auto класын ыңғайлы етеміз және ол үшін friend кілт сөзін қолданамыз.
