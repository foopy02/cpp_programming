Кездейсоқ сандар

Кездейсоқ сандарды жасау үшін rand() әдісін пайдалану керек. 1 мен 20 арасындағы кездейсоқ санды жасау үшін сізге келесі код қажет:

 int num = 1 + rand() %  20;

Бұл кодты пайдаланған кезде нөмір кездейсоқ жасалады, бірақ әрқашан бірдей болады. Бағдарлама іске қосылған сайын жаңа кездейсоқ санды көрсету үшін сіз кітапхананы пайдалануыңыз керек: #include <time.h>. Оны негізгі функцияға қосқаннан кейін келесі пәрменді жазыңыз: srand(time(NULL));﻿.


Орындау мысалы:

#include <iostream>
        #include <time.h>
        using namespace std; 
        
        int main() {
            setlocale(LC_ALL, "Russian"); // Русская локализация консоли
            srand(time(NULL)); // Генерирует случайное число, используя текущую дату как параметр
            cout << 1 + rand() % 20 << endl;
            cout << 1 + rand() % 20 << endl;
         
            cin.get();
            return 0; 
        }

Жол функциялары

Жолдармен жұмыс істеу үшін сізге #include <string> кітапханасын қосу керек. 


Пайдаланушыдан толық жолды алу үшін "getline(cin, value)" функциясын пайдаланыңыз. Функцияда value параметрі пайдаланушының мәні жазылатын айнымалының аты болып табылады.
