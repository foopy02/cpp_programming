Класс конструкторы өте ыңғайлы және сыртқы түрі қарапайым функцияларға ұқсас. Конструктордың арқасында объект жасалған кезде бірден мәндерді орнатуға болады.


Конструктордың аты сыныппен бірдей болуы керек. Төменде конструкторды құру мысалы көрсетілген:

class Book {
        private:
            int pages;
            float weight;
        public: 
            char name;
        
            void getInfoBook () {
                cout << "В книге " << name << " находиться " << pages << " страниц. " << endl;
                cout << "При этом она весит " << weight << endl;
            } 
         
            Book (int _pages, float _weight) {
                pages = _pages;
                weight = _weight;
            }
         
            Book (int _pages, float _weight, char _name) {
                pages = _pages;
                weight = _weight;
                name = _name;
            }
        };

Бір класта бірден бірнеше конструктор болуы мүмкін. Бірнеше конструкторларды жасау арқылы біз нысанды құру кезінде әртүрлі параметрлер санын бере аламыз.


Өтілген параметрлерге және олардың деректер түрлеріне байланысты қандай конструкторды пайдалану керектігін компилятор өзі анықтайды. 


Жоғарыдағы кодта айнымалылар private қатынас модификаторымен бірге келеді. Өйткені, барлық айнымалы мәндерді жеке немесе қорғалған етіп жасаған дұрыс. Айнымалыларға тек әдістер мен конструкторлар арқылы қол жеткізу керек. Өрістерге тікелей кіру жабық болуы керек.


Сонымен қатар, біз екі нысан жасап, ақпаратты көрсетеміз:

int main() {
            Book sherlock_holms(460, 1.7f); // Используем 1 конструктор
            sherlock_holms.name = 'S'; // Устанавливаем вручную значение для name
            sherlock_holms.getInfoBook ();
         
            Book green_mile(750, 2.8f, 'G'); // Используем второй конструктор
            green_mile.getInfoBook ();
         
            return 0;
        }

name айнымалысының жалпыға қолжетімді модификаторы болғандықтан, біз оған бөлек сілтеме жасай аламыз. Мұны істемей, оған жеке модификаторды да орнатқан дұрыс.


Сондай-ақ конструктор ешбір параметрді қабылдамауы мүмкін. Бұл жағдайда нысанды жасау кезінде қосымша ештеңе жіберудің қажеті жоқ.


Деструкторлар

Деструкторлар конструкторларға қарама-қарсы жұмыс істейді. Конструктор объект жасалғанда, ал жойылған кезде деструктор шақырылады.

Деструкторды құру мысалы:
class Some {
        public:
            ~Some () {
                cout << "Работа с объектом завершена.";
            }
        };
