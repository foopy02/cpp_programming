Әдістің шамадан тыс жүктелуін біз ешқашан қолмен жасамағанымызға қарамастан, бірнеше рет кездестірдік. Оны түзететін уақыт келді.


Әдістің шамадан тыс жүктелуі дегеніміз не?

Әдістердің шамадан тыс жүктелуіне байланысты біз параметрлердің басқа санын немесе басқа деректер түрін қабылдайтын және бір уақытта бірдей атқа ие болатын көптеген әдістерді жасай аламыз.


Тамаша мысал - "ранд" функциясы. Біз оған әртүрлі деректер түрлерін жібере аламыз, бірақ функция қатесіз орындалады. Бұл "rand" функциясының әртүрлі деректер түрлері үшін көптеген шамадан тыс жүктелуіне байланысты. 


Артық жүктемені қалай жасауға болады?

Артық жүктеуді жасау үшін бірдей атпен әдісті жазу керек, бірақ қабылданған параметрлердің басқа санын немесе қабылданған параметрлер үшін басқа деректер түрлерін көрсету керек.


Мысалы:

void getInfo(char some) {
        cout << some << "!";
    }
    
    int getInfo(int a, int b) {
        return a + b;
    }

Енді әдісті шақырған кезде компилятор қай параметрлер жіберілгенін түсінеді және соған байланысты әртүрлі әдістер шақырылады.
