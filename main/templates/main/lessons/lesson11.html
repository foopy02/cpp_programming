С тілінде жол нысанын жасау мүмкін болмады. Жолды құру үшін таңбалар массиві пайдаланылды - char arr[]. C++ тілінде барлығы жеңілдетілді және жолдарды жасау үшін жаңа жеке нысан енгізілді. Нысанмен қатар жолдарды өңдеуге арналған жаңа функциялар пайда болды. 


Сонымен бірге, жолдар әлі де таңбалар массиві екенін ұмытпаңыз, яғни әрбір таңбамен бір жалпы массивтің элементі сияқты жеке жұмыс істеуге болады.


Жолдармен жұмыс массивтермен жұмыс істеуден айырмашылығы жоқ. Бірақ тұтас жолды жазу үшін арнайы gets() функциясын пайдалана аламыз. 


Соңғы уақытта қауіпсізрек және пайдаланушыдан жолды алуға мүмкіндік беретін gets_s() әдісін қолданған дұрыс. 


C++ тілі кириллица таңбаларын өңдеуді қосқан жағдайда кириллицамен оңай жұмыс істей алады. Дұрыс жұмыс істеу үшін сізге "clocale" кітапханасын қосу керек, сонымен қатар main әдісінде setlocale функциясын пайдалану керек:
#include <clocale>

        using namespace std; 
        
        int main() {
            // Установка поддержки кириллицы
            setlocale(LC_CTYPE, "rus");
         
            return 0; 
        }
