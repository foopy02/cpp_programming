Массивтер ақпараттың үлкен көлемін бір жерде сақтауға мүмкіндік береді. C++ тілінде массивтердің бірнеше негізгі түрлері бар. 

Бір өлшемді массив

Массивті құру үшін деректер түрін көрсетіп, атын жазып, тік жақшаларды қою керек. Бұл кәдімгі айнымалы мәндерді жасауға өте ұқсас, бірақ бұл жерде атаудан кейін төртбұрышты жақшалар бар.


Массивтер 0-ден басталады, сондықтан индекстегі бірінші элемент - 0, екіншісі - 1 және т.б.


Массив құру мысалдары:

char stroka[2]; // Создание пустого массива
        int[] numbers; // Будет считаться ошибкой
        
        int nums[4];
        nums[0] = 1; // Добавление элементов в массив из 4 элементов
        
        int nums2[] = { 1, 2, 3, 5 }; // Присвоение всех значений сразу

Айнымалылар сияқты массив элементтерімен жұмыс істеуге болады. Біз оларды көрсете аламыз немесе олар үшін жаңа мәндер орната аламыз.

Көпөлшемді массив

Көпөлшемді массив - әрбір элементі басқа массив болатын массив. Тәжірибеде ұя салудың үшінші деңгейінен жоғары массивтер сирек пайдаланылады. Яғни, барлық элементтері басқа массивтер және барлық элементтері басқа массивтер болып табылатын массивтер іс жүзінде өте сирек кездеседі.


Біз мұндай массивтерді зерттемейміз, өйткені олардың құрылу принципі екі өлшемді массивтермен жұмыс істегендегімен бірдей.


Екі өлшемді массивтерді жасау үшін атаудан кейін қос шаршы жақшаларды пайдалану керек. Мұндай массив матрицаға ұқсайды және келесідей жазылады:

int x[3][3] = { { 0, 34, 2 }, { 3, 4, 5 } };
        cout << x[0][1]; // Выведет 34
        
        // Можно их сразу не присваивать
        char symbols[5][2];
        symbols [0][1] = 'A';

Бірінші массивтің элементтері басқа массивтер екенін көреміз. Кез келген нысанды таңдау үшін бір өлшемді массивтермен бірдей құрылымды пайдаланыңыз, тек енді бірінші массивтің де, екіншісінің де индексін көрсетіңіз:

x[0][1] = 1; // Вместо 34 теперь будет 1
