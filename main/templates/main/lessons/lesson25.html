Сынып үлгілері функция үлгілері сияқты жұмыс істейді. Айырмашылығы мынада: сынып үлгілері функцияларды емес, сыныптың үлгі құрылымын сипаттайды.


Сынып үлгісін жасау үшін template кілт сөзін пайдаланыңыз. Үлгі класын іске асыру мысалы:

template <class SomeType>
        class Test {
        private:
            SomeType value;
        public: 
            SomeType setElement(SomeType value) {
                this->value = value;
                return value;
            }
        };
Үлгі класына негізделген нысанды жасау кезінде мәндерді ғана емес, сонымен қатар деректер түрін де беру керек.

Объектіні құру мысалы:
Test<float> newObject;
        newObject.setElement(2.6);
