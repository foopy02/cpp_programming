Айнымалылар өте маңызды, себебі олар ақпаратты сақтауға және кейінірек пайдалануға мүмкіндік береді. Бастапқыда айнымалыға неліктен бірдеңе жазу керектігі анық болмауы мүмкін, егер сіз оларсыз жай ғана мәндермен жұмыс істей алсаңыз. Айнымалылар туралы түсінік сәл кейінірек, күрделірек бағдарламалар жасауды бастағанда және ақпаратты бір жерде сақтау қажет болғанда келеді.


C++ тіліндегі айнымалы түрлері айнымалы атының алдында көрсетілген. Айнымалының мазмұны нақты деректер түріне байланысты болады. Егер бүтін сандарға арналған деректер түрі бар айнымалы мән int екенін көрсетсек, оған жолды немесе нүктесі бар санды жаза алмаймыз.


В ходе программы мы можем записывать новые значения в переменную, но тип данных должен оставаться неизменным:

float some_value = 2.95;
        some_value = 6.9; // Записали новое значение
Айнымалы мәндерді құруға арналған барлық деректер түрлерін қарастырайық.

Бүтін сандар

  • short int - бүтін сандарды сақтауға арналған. -32768-ден 32767-ге дейінгі сандар диапазоны;
  • unsigned short int - оң бүтін сандарды сақтауға арналған. 0-ден 65535-ке дейінгі сандар диапазоны;
  • int - бүтін сандарды сақтауға арналған. Сан диапазоны -2 147 483 648 мен 2 147 483 647;
  • unsigned int - оң бүтін сандарды сақтауға арналған. 0-ден 4-ке дейінгі сандар диапазоны 294 967 295;
  • long int - бүтін сандарды сақтауға арналған. Сандар -9 223 372 036 854 775 808 мен 9 223 372 036 854 775 807;
  • unsigned long int - оң бүтін сандарды сақтауға арналған. Сандар диапазоны 0-ден 18-ге дейін 446 744 073 709 551 615.

Түрлер арасындағы айырмашылық тек айнымалыда сақтауға болатын сандар ауқымында. Сондай-ақ, егер сан аз болса, long int сияқты үлкен деректер түрлерін пайдаланбаңыз, себебі ауқым неғұрлым үлкен болса, белгілі бір айнымалы үшін компьютерге көбірек жад қажет.

нүктелік сандар

  • float - өзгермелі нүкте сандарын құру. Сан диапазоны -3,4*1038 және 3,4*1038;
  • double немесе өзгермелі нүкте сандарын жасау үшін long float. Сандар ±4,9*10-324 және ±1,8*10308 аралығында болады.

Басқа деректер түрлері

  • bool - логикалық деректер түрі. Шын (ақиқат) немесе жалған (жалған) мәнді жазуға арналған;
  • char - символдармен жұмыс істеуге арналған деректер түрі. Кез келген таңбаны жалғыз тырнақшаға қоюға мүмкіндік береді.
Айнымалыны жасау үшін деректер түрін көрсетіп, оны атау керек. Оған бірден мән қосуға болады:
int a = 0; // Добавление значения сразу
        float c; // Создание переменной без значения
        short int b, y = 342; // Создание нескольких переменных
Айнымалыны шақыру үшін оған оның аты бойынша сілтеме жасауға болады.

Деректерді алу

Пайдаланушыдан деректерді алу үшін пәрменді пайдалану керек cin:
cin >> some_value;
Осылайша, пайдаланушы енгізген деректер "кейбір_мән" деп аталатын айнымалыға орналастырылады.
