Функцияларды қайталанатын кодты қоюға және қажет кезде оған сілтеме жасауға болатын шағын ішкі бағдарламалар деп атауға болады. Функциялар бағдарламаларды құруды әлдеқайда жеңілдетеді, өйткені бір типтегі кодты бірнеше рет көшірудің қажеті жоқ, бірақ біз жай ғана бір жалпы функцияны пайдалана аламыз.


Көптеген адамдар функциялар мен әдістерді шатастырады және олардың арасындағы айырмашылықты түсінбейді. Іс жүзінде ешқандай айырмашылықтар жоқ, өйткені әдістер де, функциялар да бірдей. Кластардан тыс жазылатын функциялар функциялар, ал сынып ішінде жазылатын функциялар әдістер деп аталады. C++ нысанға бағытталған тіл болмағандықтан, маңызды емес болса да, функцияларды айтқан дұрыс.


Жағдай айнымалылармен бірдей. Сыныптарда айнымалылар өрістер деп аталады, ал сыныптардан тыс олар айнымалылар деп аталады.


Функцияларды жасау үшін қайтарылатын деректер түрін көрсету керек, атауды және параметрлерді көрсету керек. Функция ештеңе қайтармаған жағдайда, деректер түрі void көрсетіледі. 


Барлық функциялар main() функциясын жасамас бұрын жазылуы керек. Бұл жағдайда функцияларды шақыру кезінде сіз қателерді алмайсыз.


Сонымен қатар, функцияларды тек «негізгі» функцияның алдында жариялап, одан кейін жазуға болады.


Функция құру

Барлық деректер негізінде біздің функция келесідей болады:
void test () {
        cout << "Вывод чего-либо в консоль" << endl;
    }
Жоғарыдағы функция ешқандай параметр қабылдамайды және ештеңені қайтармайды. Ол жай ғана консольге бір сөз жазады. Функцияны әртараптандырып, параметрді қосамыз:
void test (char symbol) {
        cout << symbol << endl;
    }
Енді функция консольде көрсетілетін параметрді алады.

«main» функцияның алдында функция сипаттамасын жазу да қисынды болар еді:
void test (char symbol);
Егер функция бір нәрсені қайтаруы керек болса, біз қайтарылатын деректер түрін белгілейміз. Деректерді қайтару үшін return кілт сөзін пайдаланыңыз:
double test (double some_number) {
        some_number *= 2;
        return some_number;
    }
