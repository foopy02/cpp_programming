Динамикалық жадты бөлу өте ыңғайлы, себебі жадты бөлу арқылы біз қандай да бір нысандағы деректер үшін орынды сақтаймыз.


C++ тілінде сәйкесінше new және delete операторлары жадты бөлуге және жоюға жауапты. Олар тек динамикалық массивтерді құру үшін ғана емес, сонымен қатар объектілер мен тіпті қарапайым айнымалылар үшін жадты бөлу үшін қолданылады.

// Операция выделения памяти (new)
        float *val = new float; // Можно выделять память под любые типы данных
        // здесь val это указатель на участок памяти типа float 
        // new служит операцией выделения этой памяти
        // delete необходим для высвобождения памяти
        delete val;
Динамикалық массив бірдей құрылымды пайдаланады. Мұнда операторлар да қолданылады. new мен delete:
char *symbols = new char [25]; // выделение памяти под массив
        // Также необходимо освободить память,
        // когда массив уже не нужен
        delete [] symbols;
