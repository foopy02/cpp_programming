C++ тілінде, көптеген басқа тілдердегі сияқты, циклдердің 3 түрі бар. Олардың әрқайсысы бір рөлді орындайды, бірақ әртүрлі жазылады. Барлық үш циклды қарастырыңыз.

Цикл for

Цикл for барлық жағдайлар бір жерде жазылады, бұл көптеген жағдайларда өте ыңғайлы. Мұндай циклдің стандартты белгісі келесідей:
for (int i = 0; i < 10; i++)
        cout << i << endl;

Цикл мәлімдемесі мыналарды қамтиды: цикл айнымалысы, оның бастапқы мәні және оның ауқымы. Жоғарыдағы мысал 0-ден 10-ға дейінгі сандарды көрсетеді.


Егер цикл мәтінінде кодтың бір ғана жолы болса, онда бұйра жақшаларды өткізіп жіберуге және жазуға болмайды.

While циклі

while циклінің мәні for циклінен онша ерекшеленбейді. Айырмашылық тек циклдің жазылуында. Әзірге сізге тек шарт жазу керек, ал қалған барлық параметрлер циклден тыс жазылады:
int  i = 1; // Айнымалы мәнді жасаңыз
    while (i <= 10) { // Здесь только условие
        cout << i << endl;
        i++; // Увеличение переменной
    }

Do while циклі

Цикл жазылуы жағынан while цикліне ұқсас, бірақ ол сәл басқаша жұмыс істейді. do..while циклі бір рет жүз пайыз орындалады, содан кейін ол шартты тексереді және ол шын болса, цикл жұмысын жалғастырады:
int x = 13;
    do {
        x--;
        cout << i << endl;
    } while (x > 10);
Мысалдан көріп отырғаныңыздай, цикл бастапқыда дұрыс емес, бірақ бұл оның бір рет іске қосылуына кедергі жасамайды.

Циклге арналған операторлар

Циклдің екі негізгі операторы бар:

  • Оператор break - циклдан толық шығу үшін қызмет етеді;
  • continue операторы тек бір итерацияны өткізіп жібереді және циклден шықпайды.
