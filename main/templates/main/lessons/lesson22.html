this көрсеткішінің көмегімен деректердің қайдан келетінін дәл анықтай аламыз. Бір кластағы деректермен жұмыс істеу кезінде көбінесе сыныптың өрістеріне немесе әдістеріне сілтеме жасау қажет. Атаулардың қайшылықтарын болдырмау үшін "осы" параметрін көрсетіңіз.


Жұмыс мысалы:

class Some {
        private
            int test;
        
        public:
            void set (int test) {
                this->test = test;
            }
        };
