Кез келген C++ бағдарламалары бірнеше қажетті құрамдастардан тұрады: 

  1. Соның ішінде кітапханалар;
  2. Аттар кеңістігін көрсету;
  3. 0 мәнін қайтаратын main функциясы.

C++ тіліндегі кітапханалар

Кітапханалар қолданбаға қосымша функцияларды қосу үшін қажет. Стандартты кітапханалардың көмегімен келесі мүмкіндіктерді қосуға болады: 

  • ақпаратты экранда көрсету;
  • пайдаланушыдан деректерді алу;
  • айнымалылар жасау.


Болашақта сіз файлдармен, ағындармен, суреттермен және басқалармен жұмыс істеуге мүмкіндік беретін басқа кітапханаларды қосасыз. Барлық кітапханалар жоғарғы жағына қосылып, келесідей көрінеді:

#include <iostream> // Стандартная библиотека
		#include <string> // Служит для работы со строками
		#include <fstream> // Служит для работы с файлами
Әртүрлі операциялық жүйелерде қосылған кітапханалардың саны әртүрлі болуы мүмкін.

Аттар кеңістігі

C++ бағдарламаларында аттар кеңістігін де көрсету әдеттегідей. Бұл кодты қысқартуға және функцияларды қай кеңістіктен таңдайтынымызды жазбауға мүмкіндік береді. 
#include <iostream>
		using namespace std; // Указываем пространство имен std
		
		int main() {
			// Можем прописать просто метод cout
			// Без указания пространства имен
			// необходимо было бы писать std::cout
			cout << "Hello World";
		 
			cin.get();
			return 0; 
		}

C++ бағдарламасының өзі

Әр C++ бағдарламасы main функциясынан басталады, сондықтан бұл функция кез келген бағдарламада болуы керек. Функция әрқашан 0 мәнін қайтарады, ол бағдарламаның дұрыс орындалғанын және тоқтатылғанын көрсетеді. 


Консоль терезесінің бірден жабылуын болдырмау үшін бағдарламаның соңына пайдаланушыдан енгізуді күтетін cin.get(); қосуға болады. Сіз сондай-ақ system("pause"); пәрменін жаза аласыз, бірақ оны пайдалану үшін #include <cstdlib>﻿ кітапханасын да қамтуыңыз керек.
