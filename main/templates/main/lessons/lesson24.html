Функция үлгілері, сондай-ақ C++ тіліндегі сынып үлгілері - үлгі құрылымдарын жасауға, содан кейін деректерді беру кезінде мәндерді де, деректер түрлерін де беруге мүмкіндік беретін қуатты технологиялар.


Үлгі функциясын жасау үшін template кілт сөзін пайдалану керек және бұрыштық жақшаларда қабылданған деректер түрін көрсету керек. Бір немесе бірнеше деректер түрін қабылдай аласыз. 


Үлгі функциясын жасау мысалы:

template <typename Type1> // Указываем что это шаблон функции
        // Принимаем параметр value и его тип данных мы получим
        // также в качестве передаваемого параметра - Type1
        void funct (Type1 *value) {
            cout << value; // Просто выводим это значение
        }
Бірнеше деректер түрін беру үшін келесіні жазыңыз:
// Здесь мы получаем сразу два типа данных
        template <typename Type1, typename Type2>
        void funct (Type1 *value, Type2 *newValue) {
            // Мы можем использовать передаваемые типы в самой функции
            Type1 someNew;
            cout << newValue; // Просто выводим значение
        }
Үлгі функциясын шақыру үшін келесіні жазыңыз:
// Вызов функции с одним параметром
        funct<double>(value);
        // Вызов с несколькими параметрами
        funct<char, double>(value, oneMoreValue);
