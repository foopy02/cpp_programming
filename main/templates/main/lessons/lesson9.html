Әр айнымалы немесе нысан деректерді компьютер жадындағы белгілі бір жерде сақтайды. Біз жаңа айнымалыны жасаған сайын жадта оның жаңа мәні бар жаңа орынды жасаймыз. Неғұрлым ұяшықтар көп болса, соғұрлым компьютер жады көбірек болады.


Компьютер жадындағы мекенжай - бұл біз қол жеткізе алатын сан. Меңзер - айнымалы мәнді және нәтижесінде оның мәнін алатын жадтағы бірдей мекенжай.


Көрсеткіштермен жұмыс істеу үшін екі арнайы таңбаны пайдалану керек:& и *. Пример использования:

int t = 237; // Простая переменная
        int *p; // Создание указателя, который принимает лишь адрес другой переменной
        p = &t; // Устанавливаем адрес нашей первой переменной

Переменные t и p будут равны числу 237, так как оба ссылаются на одну ячейку. Сам же компьютер на вычислении обеих переменных потратит меньше усилий, ведь обе переменные ссылаются на одно и то же.


C++ тіліндегі сілтемелер

Сілтемелер мен көрсеткіштер ұқсас, себебі екеуінде де мән ретінде тек нысанның мекенжайы болады.


Көрсеткіш ұяшықтың мекенжайын сақтайды және егер біз осы ұяшықтың мәнін өзгерткіміз келсе, онда біз «бағыттау» операциясын орындауымыз керек:

float some = 391; // Простая переменная
        float *u = &some; // Указатель на переменную
        *u = 98; // Изменение значения переменной

Сілтемелерде мұндай ұғым жоқ, өйткені сілтемені өзгерту арқылы сіз айнымалыны да автоматты түрде өзгертесіз. Сілтемелер айнымалыға тікелей қатысты, сондықтан олардың синтаксисі қарапайым:

char symbol = 'A'; // Простая переменная
        char &ref = symbol; // Создание ссылки на переменную
        // Поскольку мы ссылаемся на переменную, то можем её использовать
        // как отдельно взятую переменную
        cout << ref << endl; // Вывод символа "А"
        ref = 'C'; // Изменение на символ "C"

Функцияларға немесе объектілерге деректерді беру кезінде сілтемелер мен көрсеткіштерді пайдалану негізделген. Сондай-ақ, бұл технологиялар бағдарлама барысында деректердің үлкен көлемін тасымалдауға өте ыңғайлы.
