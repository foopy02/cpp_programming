C++ жүйесінде көптеген кірістірілген математикалық функцияларды пайдалануға мүмкіндік беретін cmath кітапханасы бар.


Кітапхана қосылғаннан кейін синус, косинус, квадрат түбір және т.б. алу үшін функцияларды пайдалануға болады.


Күрделі есептеулерден басқа C++ барлық математикалық операцияларды орындай алады. Бұл мына сияқты операциялар:
  • алу - z = y - x;
  • қосымша - z = y + x;
  • көбейту - z = y * x;
  • бөлу - z = y / x (нөлге бөлуге болмайды);
  • бөлінгеннен кейінгі қалдық - z = y % x;

Егер операция бір айнымалыда орындалса, жазбаны қысқартуға болады:

z = z + 20;
        z += 20; // Эта запись аналогична той, что находиться выше

Санды 1-ге көбейту немесе азайту керек болса, сәйкесінше ++ және -- жаза аламыз:

z = 10;
        z += 1;
        z++; // Аналогично тому, что прописано выше
